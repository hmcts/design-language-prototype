Finding an address

This is an HMRC pattern

This pattern is used in the Personal tax account

When to use this pattern

Use this pattern to allow user to look up their address.





How to use this pattern

When finding an address for users:

  • allow them to enter their postcode with spaces
  • allow them to enter their postcode in upper cxase or lowercase
  • don't require them to know the number of the house or building
