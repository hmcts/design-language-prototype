Design patterns Templates

What is a template

A template is a basic building block for a digital service. Templates should be used to keep services looking consistent across government.

A template can have a number of regions where design patterns, components or elements can be displayed. Some template regions should only contain spesific design patterns, components or elements. For example all templates contain a header area and this area should only be used to display the header pattern.

When to use a template

Templates should be used in all services.

When to create a new template

It's unlikley that a service team will need to create a new template as the range of templates has been proven to work for all current service. If you feel that you do need to create a new template for your service you should speak to a memeber of the design language working group.
